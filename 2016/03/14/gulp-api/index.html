工作方式、流程

gulp 使用 node.js 中的 流stream ；
方式：获取 stream -> 通过 pipe() 方法将其导入到功能插件中 -> 经处理后，可继续导入到其它插件、或直接通过 dest() 写入文件。

流程： gulp.src() 获取到要处理的文件流 -> pipe() 方法将其导入到相应的 gulp 插件 -> 再通过 pipe() 导入到 gulp.dest() -> gulp.dest() 把处理后流中的内容写入到指定文件中。
