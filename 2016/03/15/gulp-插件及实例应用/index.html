安装插件

安装插件的方法为： npm instal 插件名 。

安装需要的插件：
gulp-minify-css css 压缩
gulp-concat 合并文件
gulp-uglify 压缩 js
gulp-imagemin 压缩图片
gulp-clean 删除文件
gulp-rename 重命名文件
gulp-htmlmin html压缩
